# Decisiones (ADR)

Registro ligero de decisiones tecnicas.

## 2026-05-13 - Framework y arquitectura
- Decision: usar NestJS con modulos por dominio.
- Motivo: estructura clara, DI, escalabilidad.

## 2026-05-13 - Persistencia
- Decision: TypeORM + PostgreSQL.
- Motivo: ORM tipado y relaciones simples.

## 2026-05-13 - Autenticacion
- Decision: JWT con refresh token.
- Motivo: sesiones stateless y renovacion segura.

## 2026-05-13 - Archivos CV
- Decision: Cloudinary para almacenamiento de CV.
- Motivo: storage externo y URL publica controlada.

# Get On Board UCB - Backend 



Gestión de datos para la plataforma Get On Board UCB. Arquitectura robusta diseñada para manejar roles de Admin, Empleador, Encargado y Estudiante.



##  Tech Stack



- **Framework:** [NestJS](https://nestjs.com/)

- **Runtime:** [Node.js](https://nodejs.org/)

- **Language:** [TypeScript](https://www.typescriptlang.org/)

- **Database:** [PostgreSQL](https://www.postgresql.org/)

- **Linter & Formatter:** [ESLint](https://eslint.org/) + [Prettier](https://prettier.io/)



##  Estándares de Arquitectura



Para este repositorio mantenemos el estándar de oro de NestJS: **ESLint + Prettier**.

> **¿Por qué esta dupla?** NestJS está construido para aplicaciones empresariales. Usar la configuración de fábrica garantiza un tipado estricto y un código predecible que cualquier desarrollador senior de NestJS entenderá de inmediato.



##  GitHub Flow

2. commits descriptivos siguiendo [Conventional Commits](https://www.conventionalcommits.org/).

1. Clonar el repositorio:

   ```bash

   git clone https://github.com/Matipala/GetOnBoardBackend.git

2. Instalar dependencias:

   ```bash

   npm install

3. Ejecutar el servidor:

   ```bash

   npm run start:dev
